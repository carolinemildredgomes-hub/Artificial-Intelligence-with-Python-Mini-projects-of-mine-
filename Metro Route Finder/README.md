# Metro Route Finder

## Overview

Metro Route Finder is a Python project that determines the shortest route between two metro stations using the Breadth-First Search (BFS) algorithm.

The project models metro stations as nodes in a graph and station connections as edges. Using BFS, the program guarantees the shortest route between any two stations.

## Features

* Find shortest route between stations
* Graph-based route search
* Breadth-First Search implementation
* Parent-node path reconstruction
* CSV data storage

## Concepts Used

* Graph Theory
* Breadth-First Search (BFS)
* Queues
* State Space Search
* Path Reconstruction
* Python Data Structures

## Example

Start Station: Airport

Destination: University

Output:

Airport → Central → Museum → University

Stops: 3
