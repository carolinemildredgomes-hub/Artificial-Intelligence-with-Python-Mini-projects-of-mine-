# Treasure Hunt AI

## Overview

Treasure Hunt AI is a Python project that demonstrates the Minimax algorithm through a simple turn-based strategy game.

Players compete to reach a treasure while the AI evaluates future game states and selects the optimal move.

## Features

* Turn-based strategy gameplay
* Artificial Intelligence opponent
* Minimax algorithm implementation
* Recursive game-state evaluation
* Win/Loss prediction

## Concepts Used

* Artificial Intelligence
* Minimax Algorithm
* Recursive Search
* Game Trees
* Utility Functions
* Decision Making

## How It Works

The AI explores all possible future moves and chooses the action that maximizes its chances of winning while minimizing the opponent's advantage.

Utility Values:

* Win = 1
* Draw = 0
* Loss = -1
