# Step-by-Step Explanation


## Step 1: Knowledge Representation

Medical facts are represented as symbols.

Example:

fever = True


## Step 2: Create Rules

AI stores relationships:

IF fever AND cough AND fatigue
THEN Flu


## Step 3: Patient Input

The user enters symptoms.


Example:

fever
cough
fatigue


## Step 4: Logical Inference

The AI checks every rule.

Rule:

fever + cough + fatigue

matches the patient.

Therefore:

Flu is concluded.


## Step 5: Output

The AI displays possible diseases.
